# Parcel Boilerplate

Simple boilerplate using <a href="https://v2.parceljs.org/">Parcel2</a> with basic scss and some utils.

Develop: `npm run start`<br/>
Build: `npm run build`

### Reset

- <a href="https://necolas.github.io/normalize.css/">Normalize.css</a>
- reset.css (own created)

### Variables

- Typography
- Breakpoints
- Color

### Mixin

- Media Query
